# AI Trading Dashboard - XGBoost + LSTM Ensemble

Professional AI trading dashboard for Gold (XAUUSD) / Crude Oil (CL1!) with:

- LSTM predicts future price
- XGBoost predicts Buy/Sell signals
- Ensemble combines predictions
- Backtesting: equity curve, total return, win rate
- Candlestick chart with signals

git clone https://github.com/YOUR_USERNAME/AI_Trading_Ensemble.git
